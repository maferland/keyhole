How it works
01
Agent asks
It runs keyhole NAME instead of asking you to paste a key into the chat.
02
You type it
A form opens on a random localhost URL. The value goes straight to your chosen store.
03
Agent gets a ref
It receives a one-line retrieve command — and resolves the key only at runtime.
Destinations
keychain encrypted at rest · macOS
file:/path raw value · mode 0600
env:/path NAME=value lines · 0600
Guards
✓ binds 127.0.0.1 onlyHost must be loopback
✓ random URL token per runall else 404s
✓ rejects cross-origin POSTs
✓ single-usestores once, then 409s
